css标准解决方案

2017-10-31
技术
  • css
本文总阅读量次

1.如何使用ID和类

只有在绝对确定这个元素只会出现一次的情况下，才应该使用ID。如果你认为以后可能需要相似的元素，就使用类。

2.div和span

div是对块级元素进行分组，而span是对行内元素进行分组或标识

3.伪类

:link和:visited是链接伪类，只能应用于锚元素。:hover和:active,:focus是动态伪类

4.外边距叠加

当一个元素出现在另一个元素上面时，第一个元素的底外边距与第二个元素的顶外边距发生叠加
当一个元素包含在另一个元素中时，他们的顶和或底外边距也会发生叠加
假设有一个空元素，他有外边距，但是没有边框或内边框，在这种情况下，顶外边距与底外边距就碰到了一起，他们会发生叠加
如果这个外边距碰到另一个元素的外边距，他还会发生叠加

5.相对定位

在使用相对定位时，无论是否移动，元素任然占据原来的空间。因此，移动元素会导致它覆盖其他框。

6.绝对定位

绝对定位使元素的位置与文档流无关，因此不占据空间
注意：对于定位的主要问题是要记住梅总定位的意义，相对定位时相对于元素在文档流中的初始位置，而绝对定位时相对于距离它最近的已定位祖先元素，如果不存在已定位的祖先元素，那么相对于初始包含块
